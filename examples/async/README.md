# async

A Spork project

## Getting Started

### Prerequisites

- Python 3.9+
- Spork language toolchain

### Installation

```bash
# Install dependencies
spork sync
```

### Usage

```bash
# Start the REPL
spork repl

# Run the main function
spork run

# Execute a specific file
spork src/async/core.spork
```
